clonar o repositório com: git clone <código HTTPS do repositório>
abra o git bash na pasta do projeto
use: git init
trabalhe
use: git add . - para adicionar os arquivos modificados
quando acabar use: git commit -m "<mensagem do que vc mudou>"
use: git push - para colocar no github

pronto :)


(Se tudo der errado: git push -f origin master)
